【1】charAt()

返回指定位置的字符

'ewfwef'.charAt(5) // f
'ewfwef'[5] // f

区别：

'ewfwef'.charAt(53) // “”
'ewfwef'[59] // undefined

【4】indexOf()

返回字符首次出现的位置

'abcc'.indexOf('c') // 2

【5】lastIndexOf()

上同，倒数。

【6】includes()

判断字符串中是否包含某字符串

'abcc'.includes('c') // true

【7】search()

截取和分割

  1. split
  2. substr
  3. substring
  4. slice

【1】split()

分割为数组

'abcd'.split('') // [“a”, “b”, “c”, “d”]

tips：把字符串变为数组最快的方法

'abc'.split(' ') // [‘abc’]

【2】substr()

截取（位置，个数）

'abcdefg'.substr(2,3) // “cde”

【3】substring()

截取（开始位置，结束位置）

'abcdefg'.substring(2,3) // “c”
'abcdefg'.substring(2) // “cdefg”
'abcdefg'.substring(2,-9) // “ab” 与slice的区别

【4】slice()

截取（开始位置，结束位置）

'abcdefg'.slice(2,3) // “c”
'abcdefg'.slice(2) // “cdefg”
'abcdefg'.slice(2,-9) // “” 与substring的区别

concat()

合并两个字符串

'w'.concat('e') // ‘we’

raw：模板字符串

repeat

toUpperCase()

'a'.toUpperCase() // “A”

trim()

去掉两边的空字符串

' we '.trim() // “we”
