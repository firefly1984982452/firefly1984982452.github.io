JavaScript函数调用及this参数

JS有4种方式调用函数

  • 作为一个函数(function)——fn()直接被调用
  • 作为一个方法(methods)——obj.fn()，关联在对象上调用，实现面向对象编程
  • 作为一个构造函数(constructor)——new Fn()，实例化一个新的对象
  • 通过apply或call方法调用

对应的this的指向：

  • 函数调用：window或undefined
  • 方法调用：obj对象
  • 构造函数调用：实例化的对象
  • aplly或call：第一个参数
