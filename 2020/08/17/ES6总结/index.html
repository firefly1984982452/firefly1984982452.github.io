变量的解构赋值

数组

用法

用法：只要等号两边的模式相同，左边的变量就会被赋予对应的值，解析失败返回undefined。（按次序）

默认值

默认值：数组成员严格等于===，默认值才生效

对象

使用

使用：变量与属性同名即可取到值，解析失败返回undefined

实际上let { bar } = { bar: 'bbb' };是简写，全称是：let { bar:bar } = { bar: 'bbb' };；

所以如果变量名与属性名不一致，必须写成全称：

（字义name变量接收对象里的title字段）

在解构中，左边是模式，右边是赋值。

默认值

默认值：对象属性严格等于===，默认值才生效

注意点

（1）将一个已声明的变量解构时注意不要将{}放于行首，JavaScript会解析为代码块

（2）解构允许等号左边不放置任何变量，但是无意义。

（3）数组中模式为数字时，代表下标

字符串

字符串直接转换为数组赋值

数值和布尔值

如果等号右边是数值和布尔值，会转换成对象。

用途

(1) 变换变量的值

（2）提取对象的属性

（3）遍历Map

任何部署了 Iterator 接口的对象，都可以用for…of循环遍历

（4）输入模块的指定方法

尾调用优化

尾调用是什么

某个函数的最后一步是调用另一个函数

以下三种情况，都不属于尾调用：

情况一：调用函数g之后，还有赋值操作；
情况二：调用函数g之后，还有+1的操作；
情况三：调用函数g之后，还隐式的调用了return undefined。

尾调用优化

只保留内层函数的调用帧
