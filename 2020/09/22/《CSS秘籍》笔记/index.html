width自适应关键字

  • fill-available：撑满空间，100%
  • fit-content：内容最大宽度。文字超过会换行。
  • max-content：内容最大宽度。如果文字超过显示区域了也不会换行，所以会有200%的可能。
  • min-content：内容最小宽度。比如图片是200px，文字是300px，就取200px。
