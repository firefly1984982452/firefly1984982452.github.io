用法：

全局属性

【1】id 中的#锚点

代码中有<h1 id="test">测试</h1>时，在浏览器直接后缀加上#test即可精准定位到指定 ID 地方。

【2】title

作用：鼠标悬停时有提示文字，相当于tooltip。

【3】tabindex

作用：按tab键之后按顺序遍历。

属性值：

  • 负整数：可以获取焦点（如 javaScript 中的focus()方法），但按tab键之后不会参与遍历。这个值通常是-1。
  • 0：参与遍历，如果都是 0，按顺序遍历。
  • 正整数：参与遍历，按顺序遍历。

3 个值都有的情况下的顺序：

  1. tabindex为0；
  2. 地址栏url；
  3. tabindex为正整数。

tabindex为负数时不参与

【4】accessKey

作用：使用自定义键来聚焦。

使用：

window：使用Alt + accessKey (或者 Shift + Alt + accessKey)
macbook：使用control + option + accessKey。

【6】hidden

作用：不渲染这个 DOM 元素，相当于display:none。

【7】dir

作用：dir属性表示文字的阅读方向。

有三个可能的值。

  • ltr：从左到右阅读，比如英语。
  • rtl：从右到左阅读，阿拉伯语、波斯语、希伯来语都属于这一类。
  • auto：浏览器根据内容的解析结果，自行决定。

rtl从右到左阅读时，效果相当于text-align:right，起作用的属性是：direction:rtl。

【8】contenteditable

作用：允许修改内容。

【9】spellcheck

作用：打开拼写检查。

（我在 chrome 和 firefox 下都无效，没找到原因:(）

【10】data-属性

作用：放置自定义数据。

文本标签

【1】<br>和<wbr>

<wbr>标签可以自动断行。如果一行的宽度足够，则不断行；如果宽度不够，需要断行，就在<wbr>的位置的断行。

【2】<pre>

保留换行与空格。

【3】<strong>和<b>

加粗。

【4】<em>和<i>

斜线。

【5】<sub>、<sup>和<var>

  • <sub>标签将内容变为下标，<sup>标签将内容变为上标。

  • <var>标签表示代码或数学公式的变量。

【6】<u>和<s>

  • <u>：下划线；

  • <s>：删除线。

【7】<ins>和<del>

  • <ins>：下划线；

  • <del>：删除线。

这两个标签都有以下属性。

  • cite：该属性的值是一个 URL，表示该网址可以解释本次删改。

  • datetime：表示删改发生的时间。

【8】<code>

<code>表示代码，如果需要显示多行代码，需要放在<pre>标签里面。

【9】<kbd>和<samp>

  • <kbd>：键盘代码；

  • <samp>：示例。

【10】<mark>

突出显示。

【11】<small>

最小字号显示（12px）。

【12】<time>和<data>

  • <time>：时间；

  • <data>：数据。

【13】<address>

联系方式。

【14】<abbr>

表示缩写，它的title属性可以把缩写展示全。

【15】<dfn>

术语。

【16】<ruby>

文字的语音注释。

列表标签

【1】

    有序列表。

    属性：

    【1.1】reversed：倒序

    产生的结果就是 3、2、1。

    【1.2】start：起始编号

    产生的结果就是 5、6、7。

    【1.3】type：样式

    • a：小写字母
    • A：大写字母
    • i：小写罗马数字
    • I：大写罗马数字
    • 1：整数（默认值）

    即使编号是字母，start属性也依然使用整数。

    【2】<dl>、<dt>和<dd>

    • <dl>：列表；

    • <dt>：术语名；

    • <dd>：解释。

    多个术语（<dt>）对应一个解释（<dd>），或者多个解释（<dd>）对应一个术语（<dt>），都是合法的。

    图像标签

    【1】alt

    图片的文字说明。

    图片不显示时，图片的位置会显示该文本。

    【2】width，height

    宽度和高度，单位是像素或百分比。

    【3】referrerpolicy

    设置图片的HTTP请求时的Referer的头信息。

    【4】crossorigin

    是否采用跨域的形式下载图片

    有如下属性：

    • anonymous：跨域请求不带有用户凭证（通常是 Cookie）。

    • use-credentials：跨域请求带有用户凭证。

    【5】loading

    懒加载

    有如下属性：

    • auto：默认值，相当于没有使用loading属性。

    • lazy：启动懒加载。

    • eager：立即加载资源，无论它在页面的哪个位置。

    【6】响应式-srcset

    设置不同像素时的图片地址。

    【7】响应式-sizes

    适配不同屏幕时的图片，须与srcset属性一起使用。

    【8】响应式-<picture>

    指定不同情况加载的图片。

    【9】<figure>和<figcapt>

    • <figure>：区块；

    • <figcapt>：说明。

    链接标签

    【1】<a>

    超级链接

    【1.1】href

    指向网址

    值：

    • URL；

    • 其它

    【1.1.2】邮件：mailto:

    邮件协议还允许指定其他几个邮件要素。

    • subject：主题

    • cc：抄送

    • bcc：密送

    • body：邮件内容

    【1.1.3】电话:tel:

    【1.2】download

    下载

    【2】<link>

    • 用法 1：加载CSS
    • 用法 2：加载网站的favicon图标文件

    提供不同分辨率的图标文件：

    【2.1】media

    【3.1】type

    【3.2】nomodule

    不支持 ES6 模块时，应使用nomodule属性。

    【4】<noscript>

    不支持sciprt时

    <thead>、<tbody>、<tfoot>
    <thead>、<tbody>、<tfoot>都是块级容器元素，且都是<table>的一级子元素，分别表示表头、表体和表尾。

    表单标签

    【1】<fieldset>和<legend>

    • <fieldset>：控件组；

    • <legend>：控件组的标题。

    【5】<progress>和<meter>

    • <progress>：进度；

    • <meter>：指示器。
